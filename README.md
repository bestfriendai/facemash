# FaceMash

An Expo + React Native app scaffold for FaceMash.

## Requirements

- Node.js 18+
- npm
- Expo CLI (via `npx expo`)

## Setup

```bash
npm install
```

## Run

```bash
npm run start
```

Then press:
- `i` for iOS simulator
- `a` for Android emulator
- `w` for web

## Type-check

```bash
npx tsc --noEmit
```

## Notes

- Dependencies are aligned with Expo SDK 53 via `npx expo install --fix`.
- No app-level mock data is currently present in this repository.
